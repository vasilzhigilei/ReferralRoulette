ReferralRoulette democratizes the referral sharing process. Users are able to leave their referral links here to be randomly selected to be used by others. This random referral link generation system allows for a fair and unbiased referral link sharing system, unlike link sharing on forums or websites which only promote the owner's link.

ReferralRoulette uses a random number generator in order to select a referral link. This is the most fair system for sharing referral links, as the link a user uses isn't dependent on cronological link addition to the system or any other factors.

Other websites either only promote the owner's link, don't have any structured categorization of links based on app (i.e. many 'Robinhood'), or are simply forums where users share their links, with the first few links being the only ones typically used.
ReferralRoulette is the first of it's kind, based on a community of active users.
